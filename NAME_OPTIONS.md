## Package Name Best Practices

‚úÖ **Good:**
- Lowercase with hyphens (for PyPI)
- Descriptive and searchable
- Easy to remember
- Not trademarked

‚ùå **Avoid:**
- All one word if long (hard to read)
- Similar to existing popular packages
- Generic single words (taken)
- Underscores in PyPI name (use hyphens)
